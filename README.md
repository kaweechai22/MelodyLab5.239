# MelodyLab v5.260 Doppler Physics Direct Vs

ตรวจความถูกต้องฟิสิกส์และปรับจาก v5.259:
- เปลี่ยนพารามิเตอร์จากการปรับ vₛ/v โดยตรง เป็นปรับความเร็วแหล่งกำเนิด vₛ (m/s) โดยตรง
- แอปคำนวณ vₛ/v จาก vₛ และ v เพื่อให้การเปลี่ยน v มีผลต่อ f ที่ได้ยินตามสูตรจริง
- คงสูตรแหล่งกำเนิดเคลื่อนที่ ผู้ฟังอยู่นิ่ง: f′ = f v/(v - vₛ) และ f′ = f v/(v + vₛ)
- จำกัด vₛ ไม่ให้เกิน 0.90v เพื่อไม่ให้สูตรเข้าใกล้เอกฐาน
- ทิศทางรถทำให้ผู้ฟังซ้าย/ขวาสลับความถี่ถูกต้อง
